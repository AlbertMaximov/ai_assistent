# ИИ-ассистент для компании по продаже зимней обуви

## Описание проекта
Проект направлен на создание интеллектуального помощника (ИИ-ассистента) для компании, занимающейся продажей зимней обуви. Ассистент помогает клиентам взаимодействовать с компанией через удобный и быстрый диалоговый интерфейс. Он отвечает на вопросы, используя базу знаний, записывает клиентов на встречи и интегрируется с Google Календарем для автоматизации процессов.

---

## Функционал

### 1. **Осмысленное ведение диалога**
- Ассистент способен поддерживать естественные диалоги с клиентами благодаря использованию современных нейросетевых технологий.
- Понимает контекст запросов и предоставляет релевантные ответы.

### 2. **Ответы на вопросы по базе знаний**
- Отвечает на часто задаваемые вопросы о продукции, ценах, акциях, гарантиях и условиях доставки.
- Использует структурированную базу знаний для формирования точных и информативных ответов.

### 3. **Запись клиента на встречу**
- Предоставляет возможность записаться на консультацию или примерку обуви в магазине.
- Автоматически добавляет встречу в Google Календарь для удобства как клиента, так и сотрудников.

---

## Использованные сервисы

### 1. **Qwen**
- С помощью Qwen был создан системный промпт и разработана база знаний для ассистента.
- Qwen обеспечивает высокую точность и гибкость в формировании ответов.

### 2. **Савви (Suvvy.ai)**
- Для реализации проекта был использован сервис [Савви](https://suvvy.ai/), который предоставляет платформу для создания ИИ-ассистентов.
- Платформа предлагает гибкие тарифы, зависящие от объема и сложности запросов:
  - **Простой диалог**: Быстрая обработка простых запросов без сложных вычислений.
  - **Сложный диалог**: Обработка персонализированных запросов с анализом дополнительных параметров.
  - **Индивидуальный тариф**: Для компаний с большим количеством ежедневных запросов доступны специальные условия.

---

## Преимущества проекта

- **Экономия времени**: Клиенты получают мгновенные ответы на свои вопросы без необходимости ждать ответа оператора.
- **Удобство**: Интеграция с Google Календарем позволяет автоматизировать процесс записи на встречу.
- **Масштабируемость**: ИИ-ассистент может обрабатывать большое количество запросов одновременно, что особенно полезно в пиковые сезоны.
- **Кастомизация**: База знаний и функционал ассистента могут быть адаптированы под конкретные потребности компании.

## Техническая информация

### Пример использования
1. **Клиент:** "Какие модели зимней обуви подходят для температуры ниже -20°C?"
   - **Ассистент:** "Для температур ниже -20°C мы рекомендуем модели с утеплителем Thermolite и мембраной Gore-Tex. Например: [модель X] и [модель Y]. Хотите записаться на примерку?"
   
2. **Клиент:** "Да, хочу записаться на завтра."
   - **Ассистент:** "Выберите удобное время: 10:00, 12:00 или 15:00. Встреча будет добавлена в ваш Google Календарь."

### Интеграции
- **Google Календарь**: Для автоматической записи встреч.
- **База знаний**: Для ответов на вопросы о продуктах и услугах.

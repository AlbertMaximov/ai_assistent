# Проект: "ИИ-ассистент для компании по продаже зимней обуви"

## Описание проекта

Проект направлен на создание ИИ-ассистента, который будет помогать клиентам компании по продаже зимней обуви в режиме реального времени. Ассистент способен осмысленно вести диалог, отвечать на вопросы клиентов на основе базы знаний, записывать клиентов на встречи и автоматически добавлять события в Google Календарь. Это решение позволит компании улучшить качество обслуживания, снизить нагрузку на сотрудников и повысить конверсию за счет быстрого решения запросов клиентов.

---

## Функционал

### 1. **Осмысленное ведение диалога**
   - Интеграция нейросетевой модели для создания естественного и удобного общения с клиентами.
   - Умение адаптироваться под контекст диалога, задавать уточняющие вопросы и предоставлять персонализированные ответы.
   - Обработка как простых, так и сложных запросов:
     - Простые запросы (например, "Какие размеры доступны?") обрабатываются быстро без дополнительных вычислений.
     - Сложные запросы (например, "Какую обувь выбрать для экстремальных температур?") требуют анализа базы знаний и предоставления детализированных ответов.

### 2. **Ответы на вопросы по базе знаний**
   - Создана база знаний, содержащая информацию о продуктах, услугах, политике компании, условиях доставки и возврата.
   - Ассистент использует базу знаний для точных и актуальных ответов на вопросы клиентов.

### 3. **Запись клиента на встречу**
   - Возможность записи клиента на встречу или примерку обуви через диалог с ассистентом.
   - Автоматическое добавление событий в Google Календарь для синхронизации с расписанием сотрудников компании.

---

## Используемые сервисы

### 1. **Qwen**
   - Сервис Qwen использован для:
     - Создания системного промпта, определяющего поведение ИИ-ассистента.
     - Разработки структурированной базы знаний, которая служит основой для ответов ассистента.

### 2. **Савви (Suvvy)**
   - Платформа [Савви](https://suvvy.ai/) использована для создания и настройки ИИ-ассистента:
     - Реализация диалогового интерфейса.
     - Интеграция с Google Календарем для записи встреч.
     - Настройка модели "Pay-as-you-go" для оптимизации затрат в зависимости от объема и сложности запросов.

---

## Преимущества решения

- **Экономия времени:** Ассистент обрабатывает запросы клиентов 24/7, что позволяет сотрудникам сосредоточиться на более сложных задачах.
- **Гибкость:** Модель "Pay-as-you-go" позволяет оплачивать только фактически использованные ресурсы.
- **Персонализация:** Ассистент учитывает предпочтения клиентов и предоставляет индивидуальные рекомендации.
- **Автоматизация:** Запись на встречу и добавление событий в календарь происходит без участия сотрудников.

---

## Тарифы

- **Средняя стоимость:** от 5000₽ в месяц.
- **Модель оплаты:** "Pay-as-you-go" — стоимость зависит от объема запросов и их сложности.
- **Примерная нагрузка:** ≈500 запросов от клиентов в месяц.
- **Индивидуальный тариф:** Для компаний с большим количеством ежедневных запросов предлагаются специальные условия.
